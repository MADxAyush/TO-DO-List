# To-Do List App

A simple and responsive To-Do List application built using HTML, CSS, and JavaScript.

## Features

- Add new tasks
- Mark tasks as completed
- Edit tasks by double-clicking
- Delete tasks
- Save tasks automatically using Local Storage
- Responsive design for mobile and desktop

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Browser Local Storage

## How It Works

1. Enter a task.
2. Click Add or press Enter.
3. Tick the checkbox to mark completion.
4. Double-click a task to edit it.
5. Click Delete to remove it.
6. Tasks are automatically saved in browser Local Storage.
